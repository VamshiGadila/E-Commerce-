🛍️ E-Commerce Platform
A Modern Full-Stack Shopping Experience
A complete, production-ready e-commerce application where users can shop seamlessly and admins can manage everything efficiently — built with a powerful full-stack architecture.

✨ Why This Project?
Most beginner e-commerce apps are just CRUD demos.
This project goes beyond that by implementing:

🔐 Secure JWT Authentication

🛒 Real-time Cart Management

📦 Complete Order Lifecycle

🧑‍💼 Role-based Admin Dashboard

⚡ Scalable Backend Architecture

👉 Built to simulate a real-world industry application

🧠 What You Can Do
👤 As a User
Create account & login securely

Browse and search products

Add/remove items from cart

Place orders with shipping details

Track order history

🛠️ As an Admin
Add, edit, delete products

Upload product images

View all user orders

Update order status (Pending → Delivered)

Monitor platform activity

🏗️ Tech Stack
⚙️ Backend
Java + Spring Boot

Spring Security + JWT

Hibernate (JPA)

PostgreSQL

🎨 Frontend
React + Vite

Bootstrap

Axios

Context API

🧩 System Design (Simple View)
User → React Frontend → Spring Boot API → PostgreSQL Database
⚡ Getting Started (Quick Setup)
1️⃣ Clone the Project
git clone https://github.com/VamshiGadila/E-Commerce-.git
cd ecommerce
2️⃣ Run Backend
cd backend
./mvnw spring-boot:run
3️⃣ Run Frontend
cd frontend
npm install
npm run dev
